# PR Artifacts

Sample owner: Product/Planning with role owner inputs.

Expected files for an active work unit:

- `story-pr-plan.md`
- `branch-plan.md`
- `pr-index.md`
- `review-results.md`
- `merge-checklist.md`
- `post-merge-summary.md`

`pr-index.md` should link the GitHub branch/commit/PR handoff link for each role artifact update when work leaves the current pod.
