# Role Context Fixtures

This directory contains one context fixture for each LLM SOUL role in the mobile organization.

Gatekeeper is not a role fixture. It is a non-LLM deterministic predicate surface and has no SOUL.md. Role fixtures may reference Gatekeeper only as a required check or evidence consumer.

Forbidden fixtures:

- CTO
- local-orchestrator
- Gatekeeper SOUL.md
