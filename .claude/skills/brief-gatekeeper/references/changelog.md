# Brief Gatekeeper — Changelog

## v0.13 (2026-04-29)

### 배경
v1.0 블라인드 TC 5건 전수 Fail. 결함 15건 도출 (설계 7 + 환경 2 + 해석 5).
Anthropic 공식 Agent Skills 가이드 재정독 후 패치.

### 주요 변경

**1. YAML 프론트매터 공식 스펙 준수**
- 제거: `when_to_use`, `hooks` (공식 스펙 외 필드)
- 유지·수정: `name`, `description`만
- `description`을 3인칭·what+when 구조로 재작성 (공식 베스트 프랙티스 준수)

**2. 스킵 조건을 화이트리스트로 전환 (D-1 수정)**
- 기존: "이미 분명한 요청" 같은 모호 조건 → 자의 스킵 유발
- 신규: 6개 명시 항목만 스킵 허용, 나머지는 무조건 체크리스트 수행
- **"요약·정리·해석·분석·검토·처리"는 스킵 대상이 아님**을 본문에 명시

**3. Step 4 확인을 전 분기 필수로 확정 (D-2 수정)**
- 기존: 분기 1 "바로 진행"과 Step 4 "확인 필수" 상호 충돌
- 신규: 분기 1·3·4 모두 Step 4 실행. 예외 없음을 SKILL.md에 명시

**4. 질문 도구 환경 적응 규정 추가 (E-1, E-2 수정)**
- 기존: "반드시 AskUserQuestion 툴" — 특정 환경 전용 도구로 범용성 부족
- v0.14: AskUserQuestion 필수 규정 삭제 → "구조화된 질문 포맷(번호 선택지 + ★)"으로 대체
- 신규: 환경별 도구 선택 표 추가 (Claude Code / 채팅 / API)
- 채팅 환경은 채팅 메시지 + 번호 선택지 + ★ 포맷으로 대체 허용

**5. 체크리스트 출력 의무 규정 (D-6 수정)**
- 기존: 내부 판정만 하고 숨겨도 위반 아님
- 신규: "반드시 판정표를 응답에 포함시킨다. 내부 판정만 수행하고 숨기면 Fail" 명시
- Step 2의 출력 형식 템플릿 제공

**6. "요청 동사" 구체성 기준 추가 (D-4 수정)**
- **구체 동작 (YES):** 분석, 요약, 검토, 작성, 추출, 비교, 계산, 번역, 변환, 교정, 생성, 매핑, 정규화, 분류
- **모호 동작 (NO):** 정리, 처리, 봐줘, 손봐줘, 확인해줘, 체크해봐, 이거 해줘, 어떻게 생각해, 조치해줘, 다뤄줘
- 모호 동작은 재질문 필수

**7. 5단계 필수 여부 명시 (D-3 수정)**
- 필수 수행: 분석 / 전략 제안 / 보고서 작성 / 평가·리뷰
- 스킵 허용: 단순 추출·변환·계산·번역

**8. "이미 명확한 항목은 묻지 않음" 오용 방지 (D-7 수정)**
- 추가 문구: "YES가 하나 있으니 전체 스킵은 금지"
- 나머지 NO 항목은 반드시 묻는다

**9. 표준 작업 체크리스트 블록 추가 (공식 베스트 프랙티스: Copy this checklist)**
- SKILL.md에 에이전트가 응답에 복사할 6단계 체크리스트 삽입
- 공식 "Workflows and feedback loops" 패턴 준수

**10. 태그 노출 의무화**
- `[Brief Gatekeeper] 📋` 태그로 발동 사실을 명시

## v0.15 (2026-04-29)

### 배경
Cycle 2 블라인드 TC 케이스 C(LOI 초안 + 완전 명확 지시)에서 Step 4 Fail. 분기 1(전항목 YES)에서 체크박스만 찍고 실제 확인은 스킵한 패턴 발견.

### 주요 변경

**1. Step 4 분기 1 전용 가드 추가**
- SKILL.md: "분기 1에서도 사용자 응답을 받는 단계로 실행" 명시
- "자명/명확함/추가 확인 불필요" 등으로 스킵 정당화 → Fail 명시
- 응답 없이 `[x] Step 4` 표기 → 체크리스트 위조(Fail) 명시

**2. Step 4 필수 산출물 3개 의무화**
- procedure.md: 요약 박스 + 진행 방식 리스트 + 선택지 블록 3개 필수
- 선택지 블록 없으면 자동 Fail
- Step 4 턴과 실행 턴 분리 필수

**3. Anti-Example 추가**
- examples/confirmation.md에 분기 1 Step 4 스킵 anti-example 추가
- "체크박스만 찍고 넘어가기" 패턴을 명시적으로 금지
