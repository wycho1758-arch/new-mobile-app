# Mobile App Organization Overview

This section defines the operating model for the new mobile-app development LLM organization — domain instance #1 and the reference implementation of the standard process. The organization is created via the admin-portal/admin-api creation flow and is composed of 6 LLM agents plus a non-LLM Gatekeeper (DEC-001).

It defines the organization's:
- Direction and constraints
- Roles
- Workflows
- Skills
- SOUL.md templates
- Development guidance
- Progress plan

When standing up a new domain organization, replicate this section as the pattern, but derive the actual content fresh by following the procedures in the [00] standard process.
