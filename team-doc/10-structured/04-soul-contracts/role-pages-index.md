# Role page index

The SOUL.md template page hosts a common base skeleton and an index of six role-specific child pages. Each child SOUL.md inherits the standard 8-section base (Identity, Responsibilities, Skills, Communication Style, Decision Making, Boundaries, Goals, Working Principles) and fills in only the role-specific placeholders. The non-LLM Gatekeeper has no SOUL.md and is not part of this index.

Notes:
- Gate/approval holders (Product/Planning, Mobile Architect, QA/Release) run on the supervisor model and permission profile (read/write/execute/communicate/delegate).
- Individual contributors (Design, Mobile App Dev, Backend/API Integrator) run on the IC model; implementation ICs add `execute` as a documented deviation, while non-executing Design keeps read/write/communicate.
